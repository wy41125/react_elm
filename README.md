#前言
这个项目想熟悉下react全家桶的技术框架
#技术栈
react16 + react-redux + react-router + es6 + axios + less + webpack

# 项目运行
`node >= 6.0`
```
  git clone git@github.com:liuyangjike/react-elm.git
  cd react-elm
  npm install 或用 cnpm
  npm run start
```
